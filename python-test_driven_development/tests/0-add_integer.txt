===============================
Test for add_integer function
===============================

Normal integers
---------------
>>> add_integer(1, 2)
3

Negative numbers
----------------
>>> add_integer(100, -2)
98

Default argument (b = 98)
-------------------------
>>> add_integer(2)
100

Float values
------------
>>> add_integer(100.3, -2)
98

Only floats
-----------
>>> add_integer(4.5, 2.3)
6

Error: string as second argument
--------------------------------
>>> add_integer(4, "School")
Traceback (most recent call last):
TypeError: b must be an integer

Error: None as argument
----------------------
>>> add_integer(None)
Traceback (most recent call last):
TypeError: a must be an integer
